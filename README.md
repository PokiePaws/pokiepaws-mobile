 #🐾 PokiePaws Mobile

Aplikacja mobilna dla **właścicieli zwierząt** w platformie franczyzowej PokiePaws.  
Umożliwia zarządzanie zwierzętami, umawianie wizyt w gabinetach weterynaryjnych oraz śledzenie historii leczenia.

---

## 📋 Opis

Aplikacja mobilna jest jednym z czterech komponentów systemu PokiePaws. Skierowana jest do roli **Owner (Właściciel zwierzęcia)** i komunikuje się z centralnym REST API.

### Funkcjonalności
- 📝 Rejestracja i logowanie (email + hasło)
- 🏥 Przeglądanie listy gabinetów weterynaryjnych sieci PokiePaws
- 🐶 Zarządzanie listą swoich zwierząt (imię, rasa, data ur., chip, notatki)
- 📅 Umawianie wizyt — wybór gabinetu, lekarza, daty i godziny
- 📋 Historia wizyt — nadchodzące i przeszłe
- 🔍 Szczegóły wizyty — diagnoza, zalecenia, recepty
- ❌ Anulowanie wizyty
- 🔔 Push notifications (FCM) — potwierdzenia, przypomnienia o wizytach i szczepieniach
- 🌐 Obsługa języka polskiego i angielskiego
- 📶 Tryb offline z synchronizacją po przywróceniu połączenia
